# Byultube

Youtube clone app built in Node.js
(Thumbnail)

## Technologies

- Node.js
- Pug
- Sass
- MongoDB
- Deployment : Render

## Features

- Authentication with Passport.js
- Updoad video
- Record video (web cam only)
- Cutomized video player
- Upload profile image
- Video CRUD
- Comments CRUD
- Image/Video are stored in AWS S3 bucket
